# Anirudh Portfolio — Framer-inspired Static Build

## What is included
- Home page
- Projects Index
- Individual Project Detail pages
- Industries collection + detail pages
- Reviews page (implemented as factual highlights to avoid inventing testimonials)
- FAQs page
- Legal index
- Privacy Policy
- Terms of Service
- 404 page

## Notes
This build mirrors the structure and visual rhythm of the Formix reference site, but does not copy proprietary assets or exact artwork.

## Run locally
Run from the root of this project:

```bash
python -m http.server 8000
```

Then open:
`http://localhost:8000`
